Acer 安装完 Ubuntu 开机还是进入 Win10 解决方案

发表于 2019-11-26 | |

如果你已安装完 Ubuntu 但开机还是进入 Windows 可以直接看第五条

首先吐槽一下网上不负责任的教程，不分 UEFI 引导还是 Legacy 引导就开始指导如何安装，事实上安装过程中每一步的作用他们也不见得知道。本文不是一个详细的教程，只是我的成功安装步骤的概述。
我的电脑是 Acer，已安装 Win 10，UEFI引导，GPT 分区。

安装过程

  1. 给 Linux 画出足够的空间，我清理出了一块不用的分区，直接删除卷。
  2. 关闭快速启动，关闭安全启动。这两步作用我也不清楚，似乎安全启动开着也没事。
  3. 制作系统盘，从优盘启动。

  4. 选择其它选项，手动分区。

    • Ext4 日志文件系统，30 G，挂载点 /
    • 交换空间，8 G
    • Ext4 日志文件系统，100 G，挂载点 /home

    三个分区全为逻辑分区，虽然我不明白逻辑分区和主分区的区别。没有 /boot 分区，UEFI 引导下安装过程中会自动把 grub 安装到 Windows 使用的 efi 分区中，但是为了安全起见还是手动选择一下启动引导器要安装的分区，一般应该是 /dev/sda1/，等待安装完成。

  5. 重启进入 BIOS，手动选择信任的 efi 文件，即 EFI/ubuntu/grubx64.efi，名称随便填。如果这一块是灰色的设置下 supervisor password，再不行把 secure boot 打开，设置完了再关了。再重启到 BIOS 将你设置的名称的启动顺序调到最前。再重启，完成。有的电脑不需要第五步开机就能看到 grub，但我的电脑的需要，不知是不是 Acer 的所有机器都这样。

反思

刚开始装完 Ubuntu 之后还是进入 Windows，启动顺序里也没有 Ubuntu，boot-repair 也用了，efi 分区里也有引导文件，但就是进不去以为是安装问题，又重新安装了两边之后才看到一篇帖子说要手动设置引导文件，哭
