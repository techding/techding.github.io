Colors

  • 内置颜色，多到记不住，如color： red
  • 十六进制表示，如color: #4B0082，每两位分别代表 RGB ，最大值为 255
  • RGB ，如color： rgb(0,255,0)，每个值的范围是 0 - 255
  • RGBA ，如color： rgb(0,255,0，0.9)，最后一个值控制透明度，范围 0 - 1，1 为不透明
